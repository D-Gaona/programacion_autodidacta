Instrucciones para esta Carpeta:

Esta carpeta unicamentente sera utilizada para almacenar lo apuntes generados a partir de nuestras investigaciones autodidactas. 

Para mantener un orden el formato para el nombre de cada apunte será:


"Lenguaje.Tema Central.Fuente"


Se usará la palabra "General" para referirnos a los apuntes que no tengan un lenguaje especifico y pueden ser aplicados en la generalidad de la programación.
